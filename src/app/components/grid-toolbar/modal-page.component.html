Автоматизированный сервис выявления потребностей

Обзор сервиса

Сервис предоставляет возможность построения сценария телефонного диалога с собеседником. С помощью сервеса можно озвучивать необходиму информацию и записывать ответы собеседника. Редактирование диалогов сценария происходит с использованием визуального редактора

Текст написанный в центре фигур, синтезируется и воспроизводится собеседнику и в зависимости от его ответов, происходит дальнейший переход между блоками. Надписи над стрелками указывают на ключевые слова (первое из списка) для перехода по ветке

 Создание сценария

Создание сценария происходит путем добавления новых блоков и переходов между ними.

Блоки можно добавлять с помощью иконок вокруг фигуры или через контекстное меню (нажитем правой кнопкой мыши на фигуре диалога).

Типа блоков сценария

Сервис предусматривает три речевых блока.

и системный блок, для взаимодействия с базой данных. Он не синтезирует речь.

 Редактирование

Текст для синтезирования можно вводить как в сам блок (двойной клик мыши переводит в рижим редактирования содержимое блока), так и в панели справа

Ключевые слова можно, так же добавлять на графе и в панели. Для добавления нескольких слов: на графе - слова вводятся через запятую ",", а в панели через ввод слова и нажания клавиши "Enter"

На графе отображается только первое слово из списка. При редактировании ключевых слов на графе, все слова перезаписываются

Запись ответов

Блок записи ответов может сохранять сказанное собеседником ключевое слово в заданную переменную, которую можно использовать в речи или сохранить в базу данных с помощью cистемного блока. Если ключевое слово не названо, речевой модуль повторяется или при наличии ветки с красной стрелкой, переходит к другому блоку

Если ключевые слова не указаны, то записывается любой ответ
